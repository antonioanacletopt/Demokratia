# **App Name**: Demokratia Portugal

## Core Features:

- Simulador Económico Data-Driven: Uma ferramenta que permite aos utilizadores introduzir variáveis políticas hipotéticas e visualizar o seu impacto económico simulado, utilizando dados reais de fontes portuguesas e internacionais. A ferramenta usa raciocínio para decidir quando ou se deve incorporar informações no resultado.
- Explorador de Dados Públicos: Uma interface intuitiva para explorar e compreender dados económicos e sociais em tempo real provenientes de APIs externas (Banco Mundial, Eurostat, DBnomics) e fontes de dados portuguesas.
- Perfil de Utilizador e Definições: Funcionalidades básicas de gestão de utilizadores, incluindo criação de perfil, ajustes de definições e autenticação segura.

## Style Guidelines:

- Cor primária da marca: Um indigo profundo e autoritário (#2222C3), que evoca confiança, estabilidade e inteligência.
- Cor de fundo: Um cinza frio muito subtil e dessaturado com um tom de indigo (#F0F0F4), fornecendo uma tela limpa e neutra para o conteúdo.
- Cor de destaque: Um azul celeste vibrante e claro (#4799EB), usado para calls to action, destaques e elementos interativos para significar clareza e ação.
- Fonte do título: 'Space Grotesk', uma fonte sans-serif com um carácter moderno e informado pela tecnologia, adequada para títulos e texto proeminente numa aplicação orientada por dados.
- Fonte do texto do corpo: 'Inter', uma sans-serif neutra que garante uma excelente legibilidade em diversos tamanhos de ecrã, ideal para tópicos de discussão e descrições de dados.
- Utilizar um conjunto de ícones claros, de linha ou glifo sólido, que transmitam uma sensação de profissionalismo, objetividade e acessibilidade para vários conceitos políticos e económicos.
- Empregar um layout estruturado, baseado numa grelha para visualização de dados e controlos do simulador, garantindo clareza, hierarquia lógica e capacidade de resposta entre dispositivos.
- Implementar micro-animações subtis e propositadas para transições e elementos interativos, melhorando o feedback do utilizador, especialmente dentro do simulador económico.
